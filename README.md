## Usage
Once the game is started, use the following controls to play:
- **Q and D keys**: Steer the vehicle
- **Z Key**: Accelerate
- **S Key**: Brake/Reverse
- **R Key**: Reset Course
- **Shift Key**: Boost
- **Space Key**: Use Item
